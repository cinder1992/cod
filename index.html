Cod

Python Extended IRC Services

Download as .zip Download as .tar.gz View on GitHub

Abstract

Cod is an extended IRC services package designed to solve a few accountability and convenience issues with existing IRC services packages.

Features

  • DNSBL lookups by nick or IP address
    This is useful in support for speed and accuracy of support.
  • Forwarding of non-services KILLs to the snoop channel
    This is useful in making operators more easily accountable for their actions.

Installation

  1. Extract cod to some directory
  2. Copy the example configuration and edit it to your needs, the defaults are pretty sane.
  3. Add the appropriate connect block to Charybdis, ponychat-ircd or any other ircd that uses TS6. Your mileage may vary.
  4. Change to the source directory src
  5. python cod.py
  6. Watch it link up and enjoy your new IRC services!

Usage

Commands in channels are prefixes by the prefix set in me::prefix.

List of Commands

Command IRCOP needed Usage
TEST no Prints a test message
JOIN yes Joins a channel. Optional flag "@" for joining as op
RBL yes Prints DNSBL lookup output to etc::snoopchan
MPD FIND no Searches the MPD database
MPD STATUS no Prints the status of mpd
OPNAME no Generates a fake military operation name
REHASH yes Rehashes config file from disk
DIE yes Kills Cod
